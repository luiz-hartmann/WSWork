# Desafio WSWork
================

## Considerações

- Arquitetura `MVVM-C`
- UI desenvolvida utilizando `View code`
- Gerenciador de dependências `Swift Package Manager`

## Features
- [x] Listagem de carros
- [x] Detalhes do carro
- [x] Paginação da listagem de carros
- [x] Tela de registro de usuário

## Bibliotecas externas
